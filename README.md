# arbol-bst-empresa-cpp

Descripción del proyecto: Una línea explicando que es un sistema de gestión de empleados usando un Árbol Binario de Búsqueda (BST) implementado en C++, desarrollado como práctica de la materia Estructura de Datos en la UTA.
Tecnologías: C++ estándar, compilador g++, librería <algorithm> para la función max.
Estructura del repositorio: Las carpetas que tienes (src/main.cpp, capturas/, README.md).
Cómo compilar y ejecutar: Los comandos g++ main.cpp -o arbol y ./arbol (o arbol.exe en Windows). Si quieres ser más específico, puedes mencionar que fue desarrollado con el estándar C++11 o superior: g++ -std=c++11 main.cpp -o arbol.
Funcionalidades del sistema: Las 8 opciones del menú: registrar empleado, buscar por ID, ver raíz, inorden, preorden, postorden, altura y nodos hoja.
Datos de prueba sugeridos: Los 7 empleados de la tabla (códigos 50, 30, 70, 20, 40, 60, 80) para que quien lo revise pueda probar rápidamente.
Conceptos aplicados: BST, recursividad, encapsulamiento con clase, punteros en C++.
